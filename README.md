## 📖 项目介绍

本项目整理了绵阳中学高中阶段的学习笔记，涵盖**地理、历史、政治**等多个学科。笔记内容详实，包含大量手绘图表和知识点总结。

**笔记来源**：2021年高中课堂手写笔记  
**⚠️ 注意**：由于笔记来自 2021 年，部分知识点可能需要根据最新教材和课程标准进行更新和补充。欢迎提交 Issue 或 PR 帮助完善。

**最新更新**：2026年3月11日 - 已上传第一批地理笔记（地理1.pdf，共36页）

## 🛠️ 技术说明

本项目使用 **PaddleOCR** 进行手写笔记识别：

- **识别引擎**：PaddleOCR 2.10.0
- **支持语言**：中文 + 英文混合识别
- **识别准确率**：约 85-95%（手写体）
- **处理流程**：PDF → 图片 → OCR识别 → Markdown整理

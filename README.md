# 绵阳中学高中地理笔记

[![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)

> 一份来自绵阳中学的高中地理手写笔记，开源给所有需要的学弟学妹。

## 📖 项目介绍

本项目整理了绵阳中学高中阶段的地理学习笔记，涵盖自然地理、人文地理和区域地理三大模块。笔记内容详实，包含大量手绘图表和知识点总结。

## 📂 目录结构

```
.
├── docs/                          # 整理后的笔记文档
│   ├── 01-自然地理/
│   │   ├── 01-地球与地图.md
│   │   ├── 02-大气环境.md         # ← 2.1 冷热不均引起大气运动
│   │   ├── 03-海洋环境.md
│   │   ├── 04-陆地环境.md
│   │   └── 05-自然资源.md
│   ├── 02-人文地理/
│   │   ├── 01-人口与城市.md
│   │   ├── 02-农业地域.md
│   │   ├── 03-工业地域.md
│   │   └── 04-交通与贸易.md
│   └── 03-区域地理/
│       ├── 01-世界地理.md
│       └── 02-中国地理.md
├── images/                        # 笔记中的插图、图表

└── README.md                      # 本文件
```

## 一、洋流的分类

### 按性质分类

| 类型 | 特征 | 水温对比 |
|------|------|----------|
| **暖流** | 从水温高的海区流向水温低的海区 | 比流经海区水温高 |
| **寒流** | 从水温低的海区流向水温高的海区 | 比流经海区水温低 |

### 按成因分类

| 类型 | 成因 | 举例 |
|------|------|------|
| **风海流** | 盛行风吹拂下的海水运动 | 信风带、西风带形成的洋流 |
| **密度流** | 海水密度差异引起 | 红海-亚丁湾、地中海-大西洋 |
| **补偿流** | 海水多的向海水少的方向运动 | 秘鲁寒流（上升补偿流） |

**补偿流细分**：
- 水平补偿流
- 垂直补偿流：上升补偿流、下降补偿流

### 季风洋流（北印度洋）

| 季节 | 风向 | 洋流方向 | 特点 |
|------|------|----------|------|
| **夏季** | 西南季风 | 顺时针流动 | 南赤道暖流 → 索马里寒流 |
| **冬季** | 东北季风 | 逆时针流动 | 索马里暖流 → 赤道逆流 |

## 二、洋流分布规律

### 影响因素

1. **盛行风**（主要动力）
2. **地转偏向力**
3. **陆地轮廓**

### 全球洋流分布模式

**中低纬度海区**（以副热带为中心）：
```
    北半球顺时针        南半球逆时针
    
    ① → → → → ②      ② ← ← ← ← ①
    ↑            ↓      ↓            ↑
    ④ ← ← ← ← ③      ③ → → → → ④\n    
    ① 暖流（大陆东岸/大洋西岸）
    ② 寒流（大陆西岸/大洋东岸）
    ③ 寒流
    ④ 暖流
```

**规律记忆**：
- 中低纬度：北顺南逆
- 大陆东岸（大洋西岸）：暖流
- 大陆西岸（大洋东岸）：寒流

**北半球中高纬度海区**（以副极地为中心）：
- 逆时针方向流动
- 大陆东岸：寒流
- 大陆西岸：暖流

**南半球中高纬度**：
- 形成环球性的西风漂流（寒流）
- 南极俯视图呈顺时针流动

**北印度洋**：
- 夏顺冬逆（季风洋流）

### 南极环流特点

1. 南极大陆是冰雪覆盖的大陆，影响周围气温
2. 漂浮的冰山吸收大量热量
3. 强劲的极地东风加剧海水降温

## 三、洋流对地理环境的影响

### 1. 对气候的影响

| 洋流类型 | 对沿岸气候影响 |
|----------|----------------|
| **暖流** | 增温增湿 |
| **寒流** | 降温减湿 |

**促进高低纬之间热量和水分输送和交换，维持全球热量和水分平衡。**

### 2. 对海洋生物（渔场）的影响

**渔场形成条件**：

1. **寒暖流交汇的海区**
   - 海水扰动，将海底营养盐类物质带至表层
   - 利于浮游生物大量繁殖，鱼类饵料丰富
   - 形成大渔场
   - **典型**：北海道渔场、北海渔场、纽芬兰渔场

2. **深层海水上泛（上升补偿流）**
   - 冷海水上泛，带来营养物质
   - **典型**：秘鲁渔场

### 3. 对航海的影响

| 影响 | 说明 |
|------|------|
| **有利** | 顺洋流航行，加快速度，节省燃料 |
| **不利** | 寒暖流相遇常形成大雾天气；洋流从北极地区带来冰山（春秋最危险） |

### 4. 对海洋污染的影响

- **有利**：利于污染物的扩散，加快净化速度
- **不利**：使污染范围扩大

## 四、海水的温度

### 概念

反映海水热状况的一个物理量。

### 海水热量的收支

- **收入**：主要来自太阳辐射
- **支出**：主要是海水蒸发

### 海水温度的分布规律

#### 水平分布

1. **同一海区不同季节**：夏季水温高，冬季水温低
2. **同一季节不同海区**：温度由低纬向高纬递减
3. **同一纬度**：暖流海水温度高，寒流温度低

#### 垂直分布

| 深度 | 温度变化 | 原因 |
|------|----------|------|
| **0-10m** | 迅速降低 | 太阳辐射先到海水表面 |
| **10-100m** | 快速降低 | 海水热导率很低 |
| **100m以下** | 几乎无变化 | 不受太阳辐射影响 |

## 五、海水的盐度

### 概念

单位质量海水中所含盐类物质（氯化钠、氯化镁等）的质量。

### 影响盐度的因素

| 因素 | 影响 |
|------|------|
| **降水量与蒸发量** | 外海主要影响因素 |
| **河流径流量** | 近海主要影响因素 |
| **洋流** | 同一纬度暖流盐度大于寒流 |
| **高纬度结冰/融冰** | 结冰盐度高，融冰盐度低 |
| **海域封闭程度** | 封闭程度高，盐度异常 |

### 全球盐度分布规律

**总体规律**：由副热带海区向南北两侧递减

**最高盐度海区**：红海
- 原因：
  1. 热带沙漠气候，降水量远小于蒸发量
  2. 河流径流量注入少，对海水稀释作用弱
  3. 较封闭，与外界海水交换少

**最低盐度海区**：波罗的海

**世界平均盐度**：3.5%

### 特殊海区分析

**副热带海区**：
- 受副热带高气压带控制，盛行下沉气流
- 降水量 < 蒸发量
- 盐度高

**赤道海区**：
- 受赤道低气压带控制，盛行上升气流
- 降水量 > 蒸发量
- 盐度较低

**北纬60°地区**：
- 盐度比南纬60°地区低
- 原因：北纬60°有大量陆地径流注入，对海洋起稀释作用

## 核心知识点总结

1. **洋流分类**：按性质（暖流、寒流）；按成因（风海流、密度流、补偿流）
2. **分布规律**：
   - 中低纬度：北顺南逆，东暖西寒
   - 北半球中高纬：逆时针，东寒西暖
   - 南半球中高纬：西风漂流
   - 北印度洋：夏顺冬逆
3. **洋流影响**：气候、渔场、航海、污染
4. **渔场形成**：寒暖流交汇或上升补偿流
5. **盐度分布**：副热带最高，向两侧递减

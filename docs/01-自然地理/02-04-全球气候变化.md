## 2.4 全球气候变化（第18-19页）

### 气温的特征

- 年均温的高低（全年高温/低温或日均温均大于x℃）
- 最高/低温出现的月份（常用来比较南北半球或海陆气候的差异）
- 气温年较差（一年中最高月均温减去最低月均温）

### 影响气温的因素

| 因素 | 影响 |
|------|------|
| **纬度（太阳辐射）** | 最主要因素，等温线呈东西走向（与纬线平行），向北方向递减 |
| **海陆分布** | 等温线与海岸线平行或发生弯曲，点北陆北，点南陆南 |
| **地形** | - |
| **洋流** | 等温线发生弯曲，高高低低（水平面和平面都适用） |
| **大气环流** | 等温线与海岸线平行或弯曲 |
| **人类活动** | 等温线发生闭合（热岛效应） |

### 降水的条件

- 水汽
- 空气作上升运动
- 降温
- 凝结核（尘埃、固体杂质）

### 组成气候的要素

- 气压带、风带
- 大气环流（最主要）
- 季风环流

### 影响降水的因素

- 人类活动（雨岛效应）

### 降水的特征

- 年降水量
- 季节分配
- 季节变化和年际变化：三种季风气候和热带草原气候

### 影响风的因素

- 水平气压梯度力
- 摩擦力
- 狭管效应——发生在山谷

### 影响气候的因素

- 太阳辐射
- 纬度（主要）
- 大气环流（主要）
- 海陆位置
- 洋流
- 地形（高原山地气候形成）
- 人类活动

### 气候的特征

- 气温
- 降水特征
- 水热组合状况

### 根据气温和降水的资料判断气候

**判断温度带**：
- 最冷月均温 > 15℃：热带
- 最冷月均温 0-15℃：亚热带/温带海洋性气候
- 最冷月均温 < 0℃：温带/寒带

**降水类型**：
- 年雨型：热带雨林、温带海洋性气候
- 夏雨型：季风气候、热带草原气候
- 冬雨型：地中海气候
- 少雨型：热带沙漠、温带大陆性气候

### 气候变化

**不同时期的冷暖干湿变化**：地球曾经经历了温暖期和寒冷期

- 寒冷期：冰川向低纬延伸（冰川地貌：波德平原、五大湖、挪威）
- 温暖期：冰川向高纬退缩

**趋势**：气温呈波动上升

**原因**：
- 自然原因：地球处于间冰期
- 人为原因：大气中的CO₂增多，吸收地面长波辐射而升温

**影响**：
1. 沿海低地被淹没
2. 港口设施受到破坏
3. 海水入侵，地下水质下降
4. 影响沿海水产养殖
5. 对农业的影响：
   - 利于提高农作物产量（单产增加）
   - 高纬度冻土融化，耕地面积增加（可扩大播种面积）
6. 蒸发加大，改变区域降水量和降水分布格局
7. 改变地表径流
8. 加剧水资源的不稳定性与供需矛盾

**应对措施**：
- 减少化石燃料的使用
- 多使用清洁能源
- 节能减排
- 减少废弃物的排放
- 防止森林火灾
- 保护生态环境
- <400mm种草，>400mm种树
- 大力植树种草
- 加强宣传教育
- 积极应对
- 制定相关的法律法规

## 3.1 自然界的水循环（第20-21页）

### 人类活动对水循环的影响

**破坏植被**：
- 蒸腾减少，降水减少，易发生干旱

**围湖造田**：
- 削弱湖泊对其下游河段的调节能力
- 下渗减少，地表径流增加，河流速度加快，易发生洪灾

## 4.1 营造地表形态的力量（第28-31页）

### 一、内力作用

**能量来源**：来自地球内部放射性元素衰变产生的能量

#### 岩浆活动

**概念**：岩浆（软流层）侵入岩石圈或喷出地表，冷却凝固的现象

**岩石类型**：
- 花岗岩：侵入岩，无气孔，硬度较大
- 玄武岩：喷出岩，有气孔，硬度小

**影响**：岩浆只有喷出地表能直接影响地表形态，如：火山

#### 变质作用

**概念**：岩石在高温高压条件下，岩石成分或结构发生改变，生成新的岩石

**常见变质岩**：
- 石灰岩 → 大理岩
- 页岩 → 板岩

**影响**：一般发生在地下深处，不能直接塑造地表形态

#### 地壳运动

**概念**：岩层在地表发生相对位移和变形

**类型**：
- 水平运动
- 垂直运动

**影响**：
- 形成山脉（喜马拉雅山、阿尔卑斯山）
- 形成高原、山地、平原、盆地、湖泊
- 引起地表高低起伏和海陆变迁

**板块构造学说**：

1. 全球六大板块
2. 板块边界活跃（多火山、地震）
3. 板块内部较稳定（但断裂带地壳运动活跃，多地震）

**板块运动与地貌**：

| 板块边界类型 | 地貌 |
|-------------|------|
| 大陆板块与大陆板块（碰撞） | 形成山脉、高原 |
| 大陆板块与大洋板块（碰撞） | 海沟、岛弧、海岸山脉 |
| 张裂运动（生长边界） | 裂谷、海洋 |

**实例**：
- 东非大裂谷：非洲板块内部张裂
- 红海：印度洋板块和非洲板块张裂
- 大西洋中脊：海底扩张形成

### 二、外力作用

**能量来源**：来自地球外部太阳能

**表现形式**：风化、侵蚀、搬运、堆积、固结成岩

#### 风化作用

**概念**：使岩石表面变松软的现象

**分类**：
- **物理风化**：岩石在风吹日晒的情况下发生裂开或表面松软
- **化学风化**：岩石在自然界中通过化学反应使岩石表面发生裂缝或松软
- **生物风化**：生物的根植入岩石缝隙中

#### 风力作用

**风力侵蚀**：戈壁、风蚀洼地、风蚀柱、风蚀蘑菇、雅丹地貌

**风力堆积**：黄土高原上的黄土、沙丘

**分布**：全球干旱半干旱地区

#### 流水作用

**流水侵蚀**：
- 溯源侵蚀：使河流延长（分布在河流上游较坚硬岩石处）
- 下蚀：使河谷加深，形成"V"型河谷
- 侧蚀：使河谷加宽（分布在河流中下游）

**流水堆积**：冲积扇、河漫滩平原、三角洲

## 4.2 山地的形成（第32-33页）

### 褶皱

**向斜**：储水构造，储地下水

**背斜**：储油储气，建隧道

### 断层

**地垒**：块状山地（华山、庐山、泰山）

**地堑**：谷地或低地（渭河平原、汾河谷地）

### 山地对交通运输的影响

1. **运输方式**：公路优先，铁路次之
2. **线路选址**：沿山谷或河谷分布，避开陡坡
3. **线路形态**：之字形或"8"字形
4. **技术要求**：隧道、桥梁多，成本高

## 4.3 河流地貌的发育（第34-35页）

### 河流侵蚀地貌

**河流发育初期**（一般在山区）：
- 河流落差大，流速快
- 以下蚀和溯源侵蚀为主
- 形成"V"型河谷

**河流发育中期**：
- 下蚀减弱，侧蚀增强
- 河谷展宽

### 河流堆积地貌

**冲积扇**：河流出山口处

**河漫滩平原**：河流中下游

**三角洲**：河流入海口

## 补充资料（第35-36页）

### 华西秋雨

**出现位置**：我国西南部（四川、贵州、云南等地）

**成因**：
- 秋季冷空气南下，与暖湿气流相遇
- 受地形阻挡，形成准静止锋

**特点**：
- 持续时间长（9-10月）
- 雨量不大，但持续阴雨

### 厄尔尼诺现象

**概念**：赤道附近中东部太平洋，水温异常升高的现象（11月-次年3月）

**影响**：
- 太平洋东岸：降水偏多，易产生洪涝
- 太平洋西岸：降水偏少，易产生干旱
- 秘鲁渔场：渔业资源减少

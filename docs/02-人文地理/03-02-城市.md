## 一、城市内部空间结构

### 城市土地利用和功能分区

#### 1. 住宅区

**特征**：
- 占地面积最大
- 出现高级住宅区和低级住宅区且背向发展

**位置及原因**：

| 类型 | 位置 | 原因 |
|------|------|------|
| **高级住宅区** | 城市外缘，地势较高，靠近文化区 | 环境优美，空气清新，靠近文化区，人口素质高 |
| **低级住宅区** | 城市内缘，一般靠近工业区 | 方便工人上下班、购物 |

#### 2. 商业区（CBD）

**特征**：
- 占地面积最小，呈点状或带状分布
- 经济活动最繁忙
- 建筑物高大稠密
- 内部有明显分区，人口数量昼夜差异大

**位置及原因**：
- **位置**：市中心、街角路口、交通干线两侧
- **原因**：交通便利，人口密集，市场广阔，便于商品流通

#### 3. 工业区

**特征**：
- 呈片状分布
- 不断向市区外缘移动
- 趋向于主要交通干线附近

**原因**：
- 地价便宜
- 交通便利
- 减轻对市区的污染

### 影响城市功能分区的因素

#### 经济因素（主要）

**1. 付租能力**

某种经济活动是否愿意出最高的价格买地

**2. 地租（价）——影响地租的因素**

| 因素 | 影响 |
|------|------|
| **距市中心远近** | 距市中心越近，地租越高 |
| **交通通达度** | 与地租呈正相关 |

**地租与功能区布局**：
- **A（市中心）**：最适宜布局商业区（付租能力最高）
- **B（中间地带）**：最适宜布局住宅区
- **C（城市边缘）**：最适宜布局工业区（地价低）

#### 其他因素

- **历史因素**：城市土地利用随历史变化而功能发生变化（如故宫、上海外滩）
- **社会因素**：宗教信仰、知名度（如重庆解放碑）、收入
- **行政因素**：政府规划

### 城市功能区的布局

#### 工业区（有大气污染的工厂）

**布局原则**：
- 布局在城市盛行风向的下风向的郊外
- 布局在冬夏季盛行风向垂直一侧的郊外
- 布局在最小风频的上风向

**有水污染的工厂**：布局在城区河流的下游河段

**有固体废弃物的工厂**：远离居民点和农田

#### 住宅区

布局在盛行风向的上风向，城市河流的上游河段，与工业区之间要有卫生防护带

## 二、城市的形成与发展区位因素

### 自然因素

| 因素 | 影响 |
|------|------|
| **地理位置** | - |
| **地形** | 地形平坦，土壤肥沃，农业发达；城市建筑成本较低 |
| **气候** | 城市多分布在气候温和湿润的地方，热带多分布在高原上 |
| **河流** | 沿河分布，提供生活用水和便利的交通 |
| **矿产资源** | 如大庆、匹兹堡、伯明翰 |

### 社会经济因素

- 交通
- 政治、文化

## 三、城市等级体系

**规律**：
- 城市等级越高，数目越少，服务范围越大，距离越远
- 城市等级越低，数目越多，服务范围越小，距离越近

---

## 四、城市化

### 概念

人口向城镇集聚，城市范围不断扩大，乡村变为城镇

### 表现

- 城市人口增加
- 城市用地规模扩大
- 城市人口占总人口比重上升

### 城市化进程

| 阶段 | 特点 |
|------|------|
| **初期阶段** | 城市化水平低，发展慢 |
| **中期阶段** | 城市化水平较高，发展快 |
| **后期阶段** | 城市化水平高，发展慢甚至停滞 |

### 城市化对地理环境的影响

**问题**：
- 环境污染严重
- 交通拥堵
- 住房紧张
- 就业困难

**解决措施**：
- 建立卫星城，开发新区
- 改善城市交通和居住条件
- 保护和治理城市环境

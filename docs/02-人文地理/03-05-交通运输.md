## 一、能源工业

### 能源分类

| 分类方式 | 类型 | 例子 |
|----------|------|------|
| **按是否可再生** | 可再生能源 | 太阳能、风能、水能、潮汐能、生物能、地热能 |
| | 非可再生能源 | 煤、石油、天然气、页岩气、可燃冰 |
| **按开发利用时期** | 常规能源 | 煤、石油、天然气、水能 |
| | 新能源 | 太阳能、风能、地热能、潮汐能、生物能、核能 |
| **按是否有污染** | 清洁能源 | 除煤、石油之外 |
| | 污染型能源 | 煤、石油等 |

### 我国风能丰富的条件

- 距离冬季风源地近
- 地形平坦，植被稀疏，摩擦力小
- 沿海地区，海陆热力性质差异大，风力强

## 二、交通运输线路的区位因素

### 自然因素

| 因素 | 影响 |
|------|------|
| **地形** | 地形起伏大（崎岖山区），工程量大，造价高 |
| **地质** | 位于板块交界处，地壳运动活跃，多地震；经过高山峡谷多滑坡、泥石流、塌方等地质灾害 |
| **气候** | 降水多，易出现暴雨、洪涝、暴雪等气象灾害；多大风天气，多沙尘暴；冬季严寒，工人易冻伤 |
| **河湖沼泽** | 跨越河湖沼泽多，修建桥梁多，修建成本高；经过原始森林，工人易受到毒虫猛兽的袭击 |
| **土地** | 破坏农田/占用大量耕地 |

### 社会经济因素（决定性因素）

| 因素 | 说明 |
|------|------|
| **人口城镇** | 人口城镇多，经济发达（客流量大），对交通的需求量大 |
| **资源** | 沿线自然资源丰富（矿产、旅游资源），将资源优势转变为经济优势，促进沿线经济发展 |
| **经济** | 带动沿线经济发展 |
| **政治** | 民族团结、脱贫致富、国防安全 |
| **技术** | 先进的科学技术是保证 |

## 三、交通运输点的区位因素

### 港口

**自然条件**：
- 港阔水深，避风（港湾形）
- 地形平坦，利于建港

**社会经济条件**：
- 腹地广阔，经济发达
- 城市依托

### 汽车站

- 城市交通干线两侧
- 与市内干道系统和其他对外交通有方便联系

### 航空港

- 地形平坦开阔、坡度适当
- 少云雾、暴雨、风速小
- 距城市适中
- 地质条件好

## 四、交通运输布局变化的影响

### 对聚落空间形态的影响

- 交通线发展，聚落沿交通线扩展
- 交通线衰落，聚落发展缓慢或衰落

### 对商业网点分布的影响

**交通最优原则**：
- 市区环路边缘
- 市区边缘的高速公路沿线

**市场最优原则**：
- 市中心

---

## 五、航运条件的评价

### 自然条件

- 地形平坦，水流平稳，河道易于航行
- 降水多，流量大；降水季节分配均匀，径流季节变化小
- 无结冰期，通航时间长
- 支流众多，流域面积大，通航里程长

### 社会经济条件

- 城市密集，客货运输量大
- 流域内经济发达，运输量大

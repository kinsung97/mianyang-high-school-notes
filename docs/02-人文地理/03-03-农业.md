## 一、农业区位因素

### 自然因素

| 因素 | 影响 |
|------|------|
| **气候** | 光照、热量、降水 |
| **地形** | 平原适合种植业，山地适合林业、畜牧业 |
| **土壤** | 土壤肥力影响农作物产量 |
| **水源** | 灌溉水源充足 |

### 社会经济因素

| 因素 | 影响 |
|------|------|
| **市场** | 决定农业生产的类型和规模 |
| **交通** | 扩大农产品的销售范围 |
| **劳动力** | 数量和质量 |
| **政策** | 国家宏观调控 |
| **科技** | 提高产量和品质 |

## 三、农业区位因素的发展变化

### 有利影响

- 对农产品进行深加工，延长产业链
- 带动相关产业发展
- 增加农民收入，促进区域经济发展

### 不利影响

- 水土流失，土壤肥力下降
- 水资源短缺
- 土地荒漠化（西北）
- 盐碱化
- 生物多样性减少

### 区位因素的变化及影响

| 措施 | 效果 |
|------|------|
| 大棚、温室 | 延长农产品供应时间 |
| 地膜覆盖 | 改变气温、水分 |
| 增加植被覆盖率 | 减轻水土流失 |
| 发展立体农业 | 利用地形 |
| 兴修水利、跨流域调水 | 解决水资源不足 |
| 发展节水农业 | 提高水资源利用率 |
| 培育良种 | 提高抗旱抗寒能力，改善品质 |

## 四、主要农业地域类型

### 1. 季风水田农业

**分布**：亚洲季风区（东亚、东南亚、南亚）

**特点**：
- 小农经营
- 单产高，商品率低
- 机械化水平低
- 科技水平低
- 水利工程量大

**区位条件**：
- 高温多雨的气候
- 地势平坦
- 劳动力丰富
- 种植历史悠久
- 喜食稻米

### 2. 商品谷物农业

**分布**：美国中部平原、加拿大、阿根廷、澳大利亚、乌克兰

**特点**：
- 生产规模大
- 机械化程度高
- 商品率高

**主要作物**：小麦、玉米

### 3. 大牧场放牧业

**分布**：美国、澳大利亚、阿根廷、南非

**特点**：
- 生产规模大
- 专业化程度高

**典型**：阿根廷潘帕斯草原

**发展措施**：
- 培育良种牛
- 改善交通（海上冷冻船）
- 开辟水源
- 种植饲料

### 4. 乳畜业

**分布**：北美五大湖周围、西欧、中欧、澳大利亚、新西兰

**特点**：靠近市场，饲料供应充足

**区位要求**：
- 紧邻消费市场（城市人口密集）
- 饲料充足（气候温凉湿润，适合牧草生长）

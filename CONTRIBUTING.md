# 贡献指南

感谢您对绵阳中学高中地理笔记项目的关注！

## 如何贡献

### 1. 报告错误

如果您发现笔记中有：
- 错别字
- 内容错误
- 格式问题

请提交 [Issue](https://github.com/kinsung97/mianyang-high-school-notes/issues)，并附上：
- 错误描述
- 建议的修正
- 相关页面链接

### 2. 补充内容

如果您想补充更多笔记：

1. Fork 本仓库
2. 创建您的分支 (`git checkout -b feature/add-notes`)
3. 提交更改 (`git commit -am '添加 XX 章节笔记'`)
4. 推送到分支 (`git push origin feature/add-notes`)
5. 创建 Pull Request

### 3. 提交规范

**文件命名：**
- 使用中文文件名
- 格式：`章节号-章节名.md`
- 示例：`02-大气环境.md`

**Markdown 格式：**
- 使用 `#` 表示标题层级
- 表格使用标准 Markdown 表格语法
- 图片放在 `images/` 目录，使用相对路径引用

**内容规范：**
- 保持原文意思，修正明显的 OCR 错误
- 补充必要的解释和说明
- 添加核心知识点总结
